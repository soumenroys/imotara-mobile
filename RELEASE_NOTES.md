# Imotara v1.0.2 — Release Notes

**Date:** 2026-03-15
**Build:** Android versionCode 5 · iOS buildNumber 12

---

## What's New

### Multilingual Emotion Support
- Full emotion detection across all 9 Indian languages: Hindi, Tamil, Telugu, Kannada, Malayalam, Gujarati, Punjabi, Odia, and Marathi
- CONFUSED state now accurately detected in all supported languages
- Improved anger, fear, and emotion nuance for Tamil and Hindi

### Companion & Memory
- Companion memory — the AI remembers context across your sessions
- Mood journal export — download your reflection history

### Weekly Insights & Reports
- Weekly emotional summary report with trends
- Emotion Trends tab for visual history

### Voice & Multilingual Starters
- Voice input support
- Multilingual conversation starters tailored to your language

### Crisis & Safety
- Multilingual crisis support messages
- Tap-to-call emergency contact from the crisis card

### Wellbeing
- Mood streak tracking
- Daily reminder notifications
- Breathing exercise card

### Personalisation
- Dark / Light mode
- Theme-aware colors throughout all screens

### Chat Experience
- Reactions on messages
- Export chat history
- 3-step onboarding flow for first-time users

### Sync & Storage
- Local / Cloud sync with clear mode badge
- Cloud sync enabled (free for all users in this release)

### Payments
- Razorpay integration for optional license unlock and donations
- Donations are strictly optional — no paywall, no chat gating
